# TPASCSchedule
View Badminton, Pickleball, and Rock Climbing schedules at Toronto Pan Am Sports Centre from the command line.

## Installation
### pip
```sh
pip install -r requirements.txt
```

### Usage
```sh
Usage: schedule.py option
Options:
	 -h, --help
	 -b, --badminton
	 -p, --pickleball
	 -r, --rock-climbing
```
